Module tiles

Compositing API for Terminal Glasses.

The Tiles API offers a new fundamental way of interacting with the Terminal Glasses drawing surface by allowing one to draw to subsurfaces called "tiles".

checkProperty
Contains functions for type-checking arguments. If a function expects an argument of type `typename', then checkProperty[typename](val) returns true iff `val' is of type `typename'.

Fields:

  • name A Lua string or nil.
  • number A Lua number.
  • positive_number A _non-negative_ Lua number.
  • string A Lua string.
  • boolean A Lua boolean.
  • alignment A valid pair of alignment specifiers.
  • userdata Any value.
  • function A Lua function.
  • percent A Lua number between 0 and 1 (inclusive).
  • alpha An integer between 0 and 255 (inclusive).
  • colour An integer between 0x000000 and 0xFFFFFF (inclusive).
  • gradient An integer equal to either 1 or 2.
  • simplePoint A table which has fields `x' and `y' with values which are Lua numbers.
  • complexPoint A `simplePoint' which optionally contains the fields `rgb' or `opacity' with values which are a `colour' and a `percent' respectively.
  • simplePoints An array of `simplePoint' values.
  • complexPoints An array of `complexPoint' values.
  • meta An integer between 0 and 15 (inclusive). Represents a Minecraft item meta value.
  • surface A `surface' object as exposed by the Terminal Glasses API.
  • capture A `capture' object as exposed by the Terminal Glasses API.
  • bridge A wrapped Terminal Glasses Bridge (for example, the object returned by the call peripheral.wrap(sideOfBridge)).
  • functionOrNil A Lua function or nil.
